# Case — Grupo Triunfante (Relatório Design Circuit)

**Status:** Atualizado com Estratégia Microcase (Fevereiro 2026)

Este case foi validado para demonstrar **ativação de candidatos e experiência de marca** em uma jornada omnichannel.

---

## 1. Visão Geral de Impacto (Linguagem Executiva)

- **Título:** Omnichannel rebranding para talent acquisition: Afterdoor + social → Landing
- **Subtítulo:** Ativação de candidatos e experiência de marca em jornada phygital e mobile-first.
- **Métricas:**
  - 200+ inscrições em 30 dias.
  - 64%+ conversão (QR Code → candidatura).
  - 100% das vagas estratégicas preenchidas.

## 2. A Evolução do Designer (Ponto de Vista Design Circuit)

- **Research Strategy:** O case foge do óbvio. O designer não ficou no escritório; ele foi para o ônibus (**Field Research**) validar se o QR Code era escaneável em movimento.
- **Tomada de Decisão (Decision Point):** ⭐ **MUITO FORTE.** A decisão de investir em um formato físico maior (A3) e priorizar legibilidade sobre estética decorativa demonstra maturidade de produto.
- **Equilíbrio de Stakeholders:** O designer atuou como tradutor entre o marketing (brand) e o RH (conversão), garantindo que a jornada não tivesse atritos técnicos.

## 4. Por que o "Microcase" funciona aqui?

O projeto é direto. Ele identifica um problema (vagas abertas), propõe uma hipótese (QR Code no ônibus), valida a hipótese (field testing) e entrega o resultado (vagas preenchidas). É uma narrativa limpa, sem "Frankensteins" informacionais.
